# What's For Dinner 🍳

**What's For Dinner** is an interactive, responsive web application designed to provide daily meal inspiration. The platform dynamically renders detailed recipes, including preparation times, difficulty levels, complete nutritional facts, step-by-step instructions, and personalized chef tips.



## ✨ Features
- **Dynamic Content Generation:** Users can seamlessly browse through a collection of 18+ rich recipes using the "Try Another Recipe" action.
- **Interactive UI Components:** Fully functional Bootstrap tab navigation separating Ingredients, Instructions, Nutritional Data, and Chef's Tips.
- **Smart Alerts:** Dynamic time-warning indicator that shows/hides contextually based on individual recipe properties.
- **Fully Responsive Layout:** Meticulously designed to provide an optimal viewing experience across all screen sizes (Mobile, Tablet, and Desktop).
- **Modern UI/UX Design:** Clean typography, smooth hover effects, custom gradients, and pixel-perfect card layouts.

## 🛠️ Tech Stack
- **HTML5** & **CSS3** (Custom styling & utilities)
- **Bootstrap 5** (Responsive Grid, Cards, Tabs, and Utilities)
- **Vanilla JavaScript** (DOM Manipulation, State Management, Array Methods)
- **FontAwesome** (Modern vector icons)
